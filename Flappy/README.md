GosuFlappy
==========

GosuFlappy is just a poor copy of FlappyBirds build with ruby 2.0 and Gosu Gem (https://github.com/jlnr/gosu)

##Installation

  - Clone the repository.
  - Prepare installation of Gosu Gem ( for OSX ) with `brew install sdl2 libogg libvorbis`
  - Make `bundle`

###Launch it
```sh
  ruby flappy.rb
```
